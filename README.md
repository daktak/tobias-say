tobias-say
==========

Tobias Funke Cowsay and fortune-mod addons

Install
=======
* copy tobias.cow into your cowsay/cows dir
* copy tobias and tobias.dat into your futurne dir

Runnig
======
$ fortune tobias | cowsay -f tobias
